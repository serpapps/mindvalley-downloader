## Why Mindvalley Video Downloader

Mindvalley uses HLS streaming with tokenized URLs that expire, so you cannot right-click and save a lesson. Content is served from multiple sources — Mindvalley's own player alongside embedded Vimeo, YouTube, Loom, and Wistia videos — and generic download tools cannot handle the mix. There is no built-in offline mode for course content. This extension detects every stream type on Mindvalley.com, converts it to MP4, and saves it to your computer in one click.

## FAQ

**How do I download a Mindvalley video?**
Go to any course lesson on Mindvalley.com and press play on the video. Then click the extension icon in your toolbar. The extension detects the stream and lets you select quality before downloading it as MP4.

**What types of Mindvalley content can I download?**
Any video content on Mindvalley.com — quest lessons, masterclasses, and other video content. The extension also supports embedded Vimeo, YouTube, Loom, and Wistia videos that appear on Mindvalley pages.

**Why do I need to press play first?**
Mindvalley uses HLS streaming with tokenized URLs. The video stream URL is only generated after playback starts. The extension monitors that stream request to capture the download link.

**What quality options are available?**
The extension detects all available resolutions from the HLS master playlist. For embedded platforms like Vimeo or YouTube, quality options from those platforms are shown as well.

**What format are downloaded videos?**
All videos are saved as standard MP4 files that play on any device or media player.

**Does this work on Firefox?**
Yes. It supports Chrome, Edge, Brave, Opera, Whale, Yandex, and Firefox.

**Does this work with Mindvalley's mobile app?**
No. This is a desktop browser extension. It works with Mindvalley.com accessed through a desktop browser.

**Is my data safe?**
Yes. Video processing happens entirely in your browser. Authentication uses secure OTP — no passwords stored. The extension only accesses Mindvalley.com and the embedded video platforms.
